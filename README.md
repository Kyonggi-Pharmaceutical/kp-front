# kp-front

- `.env` 파일 프로젝트 루트에 구성
```bash
REACT_APP_API_URL="http://localhost:8080"
REACT_APP_GOOGLE_CLIENT_ID="{클라이언트ID}"
```

```bash
$ npm install
$ npm start
```

## 커밋 메시지 규칙

1. 형식

   > `[타입]`: `[내용]`

2. 예시

   > docs: OO메소드 관련 설명 주석
   >
   > feature: 예약 시스템의 add()

3. 타입 종류

   > \- chore : 간단한 수정
   >
   > \- feature : 새로운 기능
   >
   > \- fix : 버그 대처
   >
   > \- refactor : 코드 수정 / 리팩터링
   >
   > \- test : 테스트 추가
   >
   > \- docs : 문서 작성
   >
   > \- design : 웹 디자인
